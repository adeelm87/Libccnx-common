bool ccnxCodecSchemaV1OptionalHeadersDecoder_Decode ( CCNxCodecTlvDecoder *  decoder,
CCNxTlvDictionary *  packetDictionary 
)

The decode a list of Version 1 optional headers.

The decoder should point to the first byte of the first optional header. The decoder will advance until the end of the buffer. It is an error for the last optional header to either go beyond the end of the decoder or for it to underrun the end of the decoder. It must exactly align.

Parameters
[in]decoderThe decoder to parse
[in]packetDictionaryThe results go directly in to the provided dictionary.
Returns
true Fully parsed interest, no errors
false Error decoding, decoder is left pointing to the first byte of the error
