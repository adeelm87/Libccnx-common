CCNxTlvDictionary *  packetDictionary 
)

The decode the validation algorithm.

The decoder should point to byte 0 of the valdiation algorithm "value"

The results are put in the provided dictionary. It is an error if the "value" does not extend to the end of the decoder.

Parameters
[in]decoderThe decoder to parse
[in]packetDictionaryThe results go directly in to the provided dictionary.
Returns
true Fully parsed, no errors
false Error decoding, decoder is left pointing to the first byte of the error

The decode the validation payload.

The decoder should point to byte 0 of the valdiation payload "value" The payload is an opaque block, so this function will just put the "value" in to the proper dictionary location. There's no real parsing.
